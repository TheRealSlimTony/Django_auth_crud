how much information can i put on a QR?

The amount of information that can be encoded in a QR code depends on the version of the QR code, which determines the size and the number of modules (black and white squares) in the code. The more modules a QR code has, the more information it can encode.

The number of modules in a QR code ranges from 21 x 21 (version 1) to 177 x 177 (version 40). The amount of information that can be encoded in a QR code is measured in characters, and it varies depending on the type of data being encoded (numeric, alphanumeric, binary, or kanji characters).

For example, a QR code version 10 (57 x 57 modules) can encode up to:

  • 7,089 numeric characters
  • 4,296 alphanumeric characters
  • 2,953 binary (8-bit) bytes
  • 1,817 kanji characters

A QR code version 40 (177 x 177 modules) can encode up to:

  • 1852 numeric characters
  • 1,109 alphanumeric characters
  • 7098 binary (8-bit) bytes
  • 4,296 kanji characters

Keep in mind that the more information you encode, the denser and more complex the QR code will be, which may affect its readability and scanning performance.
